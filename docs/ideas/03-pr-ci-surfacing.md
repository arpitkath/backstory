# PR And CI Surfacing

## Priority

P0

## Idea

Surface AI session context in pull requests through a lightweight GitHub Action or generated markdown note.

## Rationale

Reviewers should see why a change happened without leaving the PR. This also helps adoption because teammates encounter Backstory naturally during review.

## Expected Outcome

Every AI-assisted PR can include a concise context block with why, key decisions, risks, and test notes.
