# Cross-Agent Session Stitching

## Priority

P2

## Idea

Merge related sessions from different tools into one continuous task trace.

## Example Flow

- Start in Cursor.
- Continue in Claude Code.
- Patch a line manually.
- Finish in Codex.

## Rationale

Developers switch tools mid-task. A stitched task trace gives a more complete explanation than isolated per-tool sessions.
